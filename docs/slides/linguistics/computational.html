Computational morphology and syntax

A language in the finite-state automata (FSA) application is a set of expressions, each of which is built from a set of symbols from an alphabet, where an alphabet is itself a set.

Typical alphabets in speech and language processing are sets of letters, phones or words.

Regular languages are languages that can be constructed out of a finite alphabet Σ using one or more of the following operations:

  • set union
  • concatenation
  • transitive closure (Kleene star)

Regular languages can be used to describe a large number of phenomena in natural language including many morphological operations and a large set of syntactic structures.

But there are linguistic constructions that cannot be described using regular languages.

Eg. unbounded copying cannot be described in terms of regular languages or even context-free languages.

Regular languages are commonly denoted via regular expressions.

A finite-state automaton is a quintuple

M = (Q, s, F, Σ, δ)

where:

  • Q is a finite set of states
  • s is a designated initial state
  • F is a designated set of final states
  • Σ is an alphabet of symbols
  • δ is a transition relation from Q × (Σ ∪ ε) to Q, where A × B denotes the cross-product of sets A and B

The cross-product of two sets creates a set of pairs, with each member of the first set paired with each member of the second set.

For example:

{a, b} × {c, d} = {(a, c),(a, d),(b, c),(b, d)}

The transition relation δ is from state/symbol pairs to states.

Kleene’s theorem states that:

  • every regular language can be recognized by a finite-state automaton;
  • every finite-state automaton recognizes a regular language.

A (2-way) finite-state transducer is a quintuple
M = (Q, s, F, Σ × Σ, δ)
where:

  • Q is a finite set of states
  • s is a designated initial state
  • F is a designated set of final states
  • Σ is an alphabet of symbols
  • δ is a transition relation from Q × (Σ ∪ ε × Σ ∪ ε) to Q

Regular languages are closed under the following operations:

  • concatenation
  • Kleene closure
  • union
  • intersection
  • difference

Regular relations are closed under the following operations:

  • concatenation
  • Kleene closure
  • union
  • composition
  • inversion

Regular relations are not closed under:

  • intersection
  • difference
